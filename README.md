Travel-Log is a full-stack travel logging application that lets users mark, track, and document the places they have visited on an interactive map.
Users can upload photos, add travel details, and visualize their journey using location markers on a map interface.

🚀 Features
🗺️ Interactive Map – Explore and mark travel locations using MapLibre and OpenStreetMap
📍 Location Markers – Visual pointers showing all visited places
📝 Travel Entry Form – Add travel details and experiences
🖼️ Image Uploads – Upload and view photos for each location
📊 Travel Dashboard – View and manage all travel history
☁️ Cloud Database – All data stored securely in MongoDB Atlas

🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB Atlas
Maps: MapLibre, OpenStreetMap
